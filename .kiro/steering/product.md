# Product Overview

Web Query is a Flutter library for querying HTML and JSON data using a simple, intuitive syntax. It provides a unified query interface for extracting data from web pages and JSON responses.

## Core Capabilities

- CSS-like selectors for HTML elements
- JSONPath-style navigation for JSON data
- Data transformation (text manipulation, regex, variable substitution)
- Advanced filtering (include/exclude patterns)
- Query composition with fallback and required queries
- URL querying and manipulation
- Interactive UI components for data visualization

## Primary Use Cases

- Web scraping and data extraction
- API response parsing
- HTML/JSON data transformation
- URL manipulation and query parameter handling
- Interactive data exploration with visual tree views
